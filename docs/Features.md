## 📊 **技术架构**

### 🏗️ **后端技术栈**
- **框架**: Laravel 10.x
- **数据库**: MySQL 8.0
- **队列**: Laravel Queue (Database Driver)
- **缓存**: Laravel Cache
- **认证**: Laravel Sanctum
- **API**: RESTful API

### 🎨 **前端技术栈**
- **框架**: React 18.x
- **路由**: Inertia.js
- **UI**: Tailwind CSS
- **图标**: React Icons
- **国际化**: react-i18next
- **状态管理**: React Hooks

### 🤖 **Telegram 集成**
- **SDK**: irazasyed/telegram-bot-sdk
- **Webhook**: 自动消息处理
- **命令系统**: 自定义 Bot 命令
- **文件上传**: 图片和文档支持
